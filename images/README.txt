IMAGE PLACEHOLDERS
==================

Please add the following images to this folder:

1. dr-naude.jpg
   - Main photo for homepage hero
   - Recommended size: 800x1000 pixels (3:4 ratio)
   - Professional headshot or warm portrait

2. dr-naude-about.jpg
   - Photo for the About page
   - Recommended size: 600x800 pixels
   - Can be same photo or different

3. logo.png (optional)
   - If you want a logo in the CMS admin
   - Recommended size: 200x50 pixels

For blog posts and audiobooks:
- Add images to the 'uploads' subfolder
- Blog images: 1200x675 pixels (16:9 ratio)
- Audiobook covers: 800x800 pixels (square)

Image Tips:
- Use .jpg for photos (smaller file size)
- Use .png for graphics with transparency
- Optimize images for web (compress them)
- Free tool: https://squoosh.app/
